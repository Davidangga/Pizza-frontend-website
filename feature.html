Website Features

Nav Bar Scroll Animation

In desktop version, when a user scroll down the page the navbar will hide itself and will only show again if the user scroll up.

Button hover animation

Button animate when mouse hover around it

Automatic slide show

The slide show automatically change image for every 3 seconds.

Pizza card animation

Animate when mouse is hover on top of it

Pop up Register page

Instead redirecting the user to another page, when clicking register menu in the navbar the user will see a pop up page for register.

Inline error message and pop up alert

Both order form and register form use inline and pop up error messages.

Highlight menu

Highlight the menu item in the navigator corresponding to the current webpage.

Choosing cards

When user choose online payment, it allow users to choose from three types of credit card: Visa, MasterCard and American Express. Based on the type of credit card, limit the length of the credit card number, 16 digits for Visa and MasterCard, 15 digits for American Express.

Extent Form

when user choose delivery or pay online an extra fields of form will appear.

same as delivery address option

When this checkbox is checked, the fields for billing address will be filled with the delivery address automatically. If the delivery address is not completed when this checkbox is checked, display an alert or inline message: “Please enter your delivery address first”.

Disable field

When user choose pay with online, they will see card number field is disable, hovewer once they click on any card, the card number field is active again.

Disable submit button

If the form submitted is still not correct, the submit button will stays disable
